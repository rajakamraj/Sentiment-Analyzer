LLP_027_UserLogin
=================

Main.py = main program, everything runs from this
DB_Manager = used for connecting to the MySQL database. May require tweaking for setting up your IP address, username and password
Generate_FinalProgram = python script to create the Main.py file from the original raw UI.py files
run_python27.bat / run_python33.bat = used on windows machines to call main.py using different versions of python

Login_Dialog_RAW.py / NewUser_Dialog_RAW.py = Freshly created python files from the UI files
Login_Dialog.ui / NewUser_Dialog.ui = user interfaces created within Qt Designer
